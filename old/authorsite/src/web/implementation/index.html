bibliographic application - implementation

analysis

The bibliographic application is sophisticated both in its concept and in its implementation in code. It is posited on the notion that a bibliography is not simply a list, but rather a network of interrelated meta-data, as explained in our Conceptual Analysis.

design

In order to facilitate an open, scalable and flexible platform, the application has been implemented as a J2EE Java application, as the Architecture and Design analysis explains. The document also discusses the design patterns used.

code

For developers, javadoc is available, as is java source cross reference. SQL, XSLT and ant build scripts are available with the source release.

deployment

Finally, we give detailed instructions on how the web application might be deployed, using the deployed demonstration application as an example.
